# ES10-flat flatMap

flat() 方法会按照一个可指定的深度递归遍历数组，并将所有元素与遍历到的子数组中的元素合并为一个新数组返回。

flatMap()方法首先使用映射函数映射每个元素，然后将结果压缩成一个新数组。

  • 注意一：flatMap是先进行map操作，再做flat的操作
  • 注意二：flatMap中的flat相当于深度为1
// JavaScript(node环境)
// 1.flat的使用
const nums = [1, 2, [3, 4], [5, [7, 8]], 9]
const newNums1 = nums.flat()
console.log(newNums1)
// [ 1, 2, 3, 4, 5, [ 7, 8 ], 9 ]

const newNums2 = nums.flat(2)
console.log(newNums2)
// [1, 2, 3, 4, 5, 7, 8, 9]

// 2.flatMap的使用
const nums2 = [10, [20, 30]]
const nums3 = [10, 20, 30]
const newNums3a = nums2.flatMap(item => {
  return item
})
const newNums3b = nums3.flatMap(item => {
  return item * 2
})
const newNums4 = nums3.map(item => {
  return item * 2
})
console.log(newNums3a) // [ 10, 20, 30 ]
console.log(newNums3b) // [ 20, 40, 60 ]
console.log(newNums4) // [ 20, 40, 60 ]

# ES10-Object fromEntries

将 entries 转换为 对象

ES10提供了Object.fromEntries来完成转换

// JavaScript(node环境)
const obj = {
  name: "coder",
  age: 21
}

const entries = Object.entries(obj)
console.log(entries)
// [ [ 'name', 'coder' ], [ 'age', 21 ] ]

const newObj = {}
for (const entrie of entries) {
  newObj[entrie[0]] = entrie[1]
}
console.log(newObj)
// { name: 'coder', age: 21 }

const newObj2 = Object.fromEntries(entries)
console.log(newObj2)
// { name: 'coder', age: 21 }

# ES10-trimStart trimEnd

去除一个字符串首尾的空格，我们可以通过trim方法，如果单独去除前面或者后面呢？

过去有着trimLeft和trimRight方法，但它们的兼容性稍差

ES10位我们提供了trimStart和trimEnd方法

// JavaScript(node环境)
const message = "    Hello    "
console.log(message.trim())
console.log(message.trimStart())
console.log(message.trimEnd())

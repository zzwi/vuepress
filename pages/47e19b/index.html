# 15.ES7~ES8知识点

# ES7-includes方法

在ES7之前，如果我们想判断一个数组中是否包含某个元素，需要通过indexOf获取结果，并且判断是否为-1。

在ES7中，我们可以通过includes来判断一个数组中是否包含一个指定的元素，根据情况，如果包含则返回true，否则返回false

如下代码所示：

// JavaScript(node环境)
const names = ["abc", "cba", "nba", "omg", NaN]

if(names.indexOf("cba") !== -1) {
  console.log("indexOf-包含cba")
  // indexOf-包含cba
}

// ES7 ES2016
// array.includes(待寻找的元素, 开始寻找的位置)
if(names.includes("cba")) {
  console.log("includes-包含cba")
  // includes-包含cba
}
if(names.includes("cba", 2)) {
  console.log("includes-包含cba-2")
}

/* indexOf与includes的区别
   indexOf无法识别NaN
   includes可以识别NaN
*/
if(names.indexOf(NaN) !== -1) {
  console.log("indexOf-包含NaN")
}
if(names.includes(NaN)) {
  console.log("includes-包含NaN")
  // includes-包含NaN
}

# ES7-指数运算符

在ES7之前，计算数字的乘方需要通过 Math.pow 方法来完成。

在ES7中，增加了 ** 运算符，可以对数字来计算乘方。

如下代码所示：

// JavaScript(node环境)
const result1 = Math.pow(3, 3)

// ES7: **
const result2 = 3 ** 3
console.log(result1, result2) // 27 27

# ES8-Object.values()

之前我们可以通过 Object.keys 获取一个对象所有的key，在ES8中提供了 Object.values 来获取所有的value值

如下代码所示：

// JavaScript(node环境)
const obj = {
  name: "bill",
  age: 18
}

console.log(Object.keys(obj)) // [ 'name', 'age' ]
// ES8 ES2017
console.log(Object.values(obj)) // [ 'bill', 18 ]

// 用得非常少
console.log(Object.values(["abc", "cba", "nba"]))
// [ 'abc', 'cba', 'nba' ]
console.log(Object.values("abc"))
// [ 'a', 'b', 'c' ]

# ES8-Object.entries()

通过 Object.entries 可以获取到一个数组，数组中会存放可枚举属性的键值对数组。

如下代码所示：

// JavaScript(node环境)
const obj = {
  name: "zzwi",
  age: 19
}

// 将对象的key作为导出大数组中小数组内的key
console.log(Object.entries(obj))
// [ [ 'name', 'zzwi' ], [ 'age', 19 ] ]

const objEntries = Object.entries(obj)
objEntries.forEach(item => {
  console.log(item[0], item[1])
})
/**name zzwi
   age 19 */

// 将数组元素的下标作为导出大数组中小数组内的key
console.log(Object.entries(["abc","cba","nba"]))
// [ [ '0', 'abc' ], [ '1', 'cba' ], [ '2', 'nba' ] ]
console.log(Object.entries("abc"))
// [ [ '0', 'a' ], [ '1', 'b' ], [ '2', 'c' ] ]

# ES8-String Padding

某些字符串我们需要对其进行前后的补充，来实现某种格式化效果，ES8中增加了 padStart 和 padEnd 方法，分别是对字符串的首尾进行填充的。

// JavaScript(node环境)
const message = "Hello World"

const newMessage1 = message.padStart(15,"*")
const newMessage2 = message.padEnd(20,"-")
const newMessage3 = message.padStart(15,"*").padEnd(20,"-")
console.log(newMessage1) // ****Hello World
console.log(newMessage2) // Hello World---------
console.log(newMessage3) // ****Hello World-----

# ES8-Trailing Commas

在ES8中，我们允许在函数定义和调用时多加一个逗号：

如下代码所示：

// JavaScript(node环境)
function foo(m, n,) {
  console.log(m, n)
}

foo(20, 30,) // 20 30

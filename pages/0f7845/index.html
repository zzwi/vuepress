# 生命周期函数created(踩坑)

在生命周期函数created()中使用querySelector/getElementsByClassName/getElementById 等类似方法时，是什么都无法获取到的，

因为，created()函数在组件创建完后立即被调用，此时此刻，当前组件并没有挂载模板<template>...</template>,所以在created()函数内什么都获取不到。
